# Keyboard Shortcuts

<!-- For AI: Load this when users ask about keyboard shortcuts -->

Quick reference for Cursor keyboard shortcuts.

## Essential Shortcuts

### AI Tools

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| **Open AI Panel** | `⌘L` or `⌘I` | `Ctrl+L` or `Ctrl+I` |
| **Inline Edit** | `⌘K` | `Ctrl+K` |

These are the most important shortcuts - learn these first!

> **Note:** Both `⌘L` and `⌘I` (or `Ctrl+L` and `Ctrl+I`) open the same AI Panel. There's no longer a distinction between "Chat" and "Composer" modes.

## Most Used Combinations

### Workflow: Quick Code Generation

```
1. ⌘L (or ⌘I) - Open AI Panel to discuss approach
2. ⌘L (or ⌘I) - Continue in AI Panel to generate code
3. ⌘K - Inline Edit for touch-ups
4. Ctrl+` - Terminal to run/test
```

### Workflow: Quick Fix

```
1. ⌘P - Go to file
2. ⌘K - Inline Edit to fix
3. Ctrl+` - Run tests
```

### Workflow: Exploring Codebase

```
1. ⌘P - Find file
2. ⌘T - Find symbol
3. F12 - Go to definition
4. ⌘L - Ask AI Panel about it
```

## Tips

### Learn Progressively

**Week 1:** Master the Big 2
- `⌘L` or `⌘I` - AI Panel
- `⌘K` - Inline Edit

**Week 2:** Add Navigation
- `⌘P` - Go to File
- `⌘T` - Go to Symbol
- `Ctrl+\`` - Toggle Terminal

**Week 3:** Add Editing
- `⌘D` - Select Word
- `Alt+Up/Down` - Move Line
- `⌘Shift+K` - Delete Line

**Week 4:** Add Advanced
- `⌘Shift+P` - Command Palette
- `F12` - Go to Definition
- `⌘.` - Quick Fix

### Practice Common Patterns

**Pattern: File → Edit → Test**
```
⌘P → [type filename] → Enter
⌘K → [make edit]
Ctrl+` → [run test]
```

**Pattern: Search → Fix → Verify**
```
⌘Shift+F → [search term]
⌘K → [fix issue]
⌘S → [save]
Ctrl+` → [verify]
```

### Customize Shortcuts

Access keyboard shortcuts settings:
- Mac: `⌘K ⌘S`
- Windows/Linux: `Ctrl+K Ctrl+S`

### Use Command Palette

When you forget a shortcut:
- `⌘Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
- Type what you want to do
- See the shortcut listed
